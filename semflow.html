PRINCIPAIS COMANDOS SEM FLOW PREDEFINIDO

COMANDO FUNÇÃO
git init Cria um novo repositório no local atual
git clone (URL do Repositório) Clona um repositório já existente para o local atual
git add (nome do arquivo) Envia as alterações do arquivo específico para área de preparação aguardando o commit
git commit -m "Mensagem do commit" Registra as alterações realizadas e acrescenta uma mensagem descrevendo informações sobre a alteração
git status Mostra a status das alterações no local de trabalho
git log Mostra um histórico de commits já realizados
git diff Exibe a diferença entre o trabalho atual e a ultima versão enviada
git tag Exibe todas as tags já criadas
git tag -a (versão) -m "Mensagem da tag" Cria uma nova tag onde é possivel indicar uma versão para ela e descrever uma mensagem descritiva
git branch Exibe uma lista das branches do repositório
git checkout -b (nome da branch) Cria uma nova branch e já muda para ela
git merge (nome da branch) Mescla as alterações da branch atual com a principal
git remote add origin (URL do repositório remoto) Adiciona um repositório em local remoto
git push origin (nome da branch) Envia todas as alterações realizadas para o repositório remoto
git pull origin (nome da branch) Atualiza o repositório local com as alterações do repositório remoto
git fetch Recupera informações do repositório remoto sem fazer merge
git merge origin/(nome da branch) Mescla as alterações remotas na branch local
git mergetool Ferramenta para ajuda na resolução de conflitos de merge
