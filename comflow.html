PRINCIPAIS COMANDOS COM FLOW PREDEFINIDO

COMANDO FUNÇÃO
git flow init Iniciar o uso do git flow no reposirório atual
FUNCIONALIDADES / FEATURES
git flow feature start MYFEATURE Cria uma nova branch da funcionalidade específica e ja altera para ela
git flow feature finish MYFEATURE Mescla as alterações desta branch para a principal, remove a brach da funcionalidade e retorna para a branch principal (develop)
git flow feature publish MYFEATURE Envia a funcionalidade criada para o servidor remoto.
git flow feature pull MYFEATURE Obtenha uma funcionalidade criada e publicada por outro usuário e que esteja em local remoto
CRIAR UMA VERSÃO / RELEASE
git flow relase start RELEASE [BASE] Começa uma nova versão criando um branch baseado no develop
git flow publish RELEASE Publica o branch da versão permitindo commits de outros desenvolvedores
git flow release track RELEASE Acompanha uma versão remota
git flow release finish RELEASE Ao final da versão, mescla o branch da versão com o master, etiqueta com seu nome a versão criada, mescla o branch de volta no develop e remove o branch da versão
HOTFIXES
git flow hotfix start VERSION [BASENAME] Inicia o hotfix onde o VERSION é o nome do novo hotfix
git flow hotfix finish VERSION Ao ser finalizado o hotfix, será mesclado na brach develop e master
